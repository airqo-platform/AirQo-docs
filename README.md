# AirQo-docs

This repo contains the AirQo documentation for the software and hardware platforms. The AirQo organisation has four main repositories:

- [AirQo-frontend](https://github.com/airqo-platform/AirQo-frontend)
- [AirQo-api](https://github.com/airqo-platform/AirQo-api)
- [AirQo-modules](https://github.com/airqo-platform/AirQo-modules)
- [AirQo-docs](https://github.com/airqo-platform/AirQo-docs)

## Contributing

Please see our [contributing guidelines](contributing.md) for information on how to contribute.

## API Documentation

The AirQo platform can be accessed programatically via an API ([API documentation](https://docs.airqo.net/airqo-platform-api/)).

## User documentation

[User documentation](https://docs.airqo.net/airqo-handbook/-MHlrqORW-vI38ybYLVC/) explains how to use AirQo platform instances (e.g. http://platform.airqo.net/, http://staging-platform.airqo.net/) via the web GUI.

## Security

For information on security considerations when developing or configuring infrastrucure, please see [security](security.md).
